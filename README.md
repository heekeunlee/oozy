# 우지커피 전국 매장 분포

우지커피 공식 매장찾기 데이터를 기반으로 전국 매장 좌표와 기간별 성장 추이를 모바일 우선 UI로 시각화한 정적 사이트입니다.

- 공식 매장찾기 좌표 447개 사용
- 2020년부터 2026년 4월 28일까지 공개 누적 수 기준 기간 필터 제공
- 브랜드 1호점 권역과 최근 등록 지점 표시

## 데이터 갱신

```bash
python3 scripts/fetch_stores.py
```

매장별 정확한 개점일은 공개 목록에 포함되어 있지 않아, 개별 매장의 연도는 공식 매장찾기 등록순을 공개 누적 수에 맞춰 배분한 추정값입니다.
